# 🦞 ClawdyClaw

Official OpenClaw Templates & Packages

Premium starter kits, skill packs, and deployment packages for your OpenClaw AI assistant. Built by the community, for the community.

## Templates

| Package | Description | Price |
|---------|-------------|-------|
| 🥚 Egg Starter | Basic SOUL.md, AGENTS.md, and essential configs | Free |
| 🦞 Molty Pack | Advanced personality templates, memory system, custom skills | $9 |
| 👑 Claw Commander | Full enterprise suite with Pretext Demo Pack | $29 |

## Features

- 🎯 SOUL.md Templates - Pre-crafted identity files
- 🛠️ Skill Packs - Ready-to-use skills for email, calendar, GitHub
- 📦 Channel Configs - Telegram, Discord, WhatsApp, Slack
- 🚀 Deploy Scripts - One-command deployment
